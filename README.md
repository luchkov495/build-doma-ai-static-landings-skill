# build-doma-ai-static-landings

Codex skill for building and reviewing static DOMA.ai-style landing pages for ЖКХ products.

## What is inside

- `SKILL.md` - skill instructions and workflow.
- `references/brand-static-landing-guide.md` - DOMA.ai landing page style guide.
- `references/screenshots/` - desktop and mobile visual references.
- `references/audits/` - computed style and layout audit data.
- `assets/static-landing-template/` - static HTML/CSS starter template.
- `agents/openai.yaml` - agent configuration.

## Install

Clone this repository into your Codex skills folder:

```bash
mkdir -p ~/.codex/skills
git clone https://github.com/aluchkov495/build-doma-ai-static-landings-skill.git ~/.codex/skills/build-doma-ai-static-landings
```

Restart Codex after cloning if the skill does not appear immediately.

## Use

Ask Codex to use `build-doma-ai-static-landings` when creating or reviewing static DOMA.ai-style landing pages.
